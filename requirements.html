Installation:

Before you start...

You will need to have Python 3 installed and know how to install pip packages.

Some useful vids:

  • Install Python 3 on Linux and Mac
  • Install Python 3 on Windows
  • Pip packages on Windows
  • Pip packages on Mac
  • Pip packages on Linux

If these do not explain all that you need to know then the internet is your best friend and can explain it better than I can!

Brief run-through

  1. Hit the download button on this website.
  2. Extract the files from the .zip file to a folder.
  3. Open your terminal/CMD and navigate to the folder with the extracted files in using "cd Folder/Location/Instead/of/this"
  4. Type "pip install -r requirements.txt". Close the terminal/CMD when it is done.
  5. Go into the folder with the extracted files in and click on main.pyw
  6. If everything has gone well, the app should open. You can create a shortcut for the file but DO NOT delete any files in the extracted folder! Enjoy:)
